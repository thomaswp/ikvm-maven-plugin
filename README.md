# ikvm-maven-plugin

This project is a fork of [samskiver's ikvm plugin](https://github.com/samskivert/ikvm-maven-plugin) for use with the [PlayN-XNA project](https://github.com/thomaswp/playn-xna/). This plugin generates CLR libraries from your project's core code and dependencies.
